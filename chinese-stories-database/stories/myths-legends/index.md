# ✨ 中国神话传说集

## 系列说明

中国神话传说是中华民族的瑰宝，讲述了天地开辟、人类起源、英雄创世的故事。这些故事充满了想象力和智慧，展现了古代人民对世界的理解和对美好生活的向往。

## 阅读建议

1. **按顺序阅读**：从创世神话开始，了解世界的起源
2. **联系生活**：结合生活中的事物，理解神话的智慧
3. **想象创造**：鼓励孩子用自己的想象续写神话
4. **文化延伸**：了解神话中提到的地名、节日等
