# 🏔️ 山海经故事集

## 系列说明

《山海经》是中国最古老的地理志和神话集，记载了上古时期的神话传说、奇珍异兽、山川地理。这里的故事充满了想象力和神秘色彩，展现了古代先民对世界的理解。

## 阅读建议

1. **先看插图**：山海经的神兽非常有趣，可以先看看插图
2. **发挥想象**：让孩子想象这些神兽的样子
3. **联系现实**：有些神兽在现实中能找到影子
4. **文化延伸**：了解神兽在后世文化中的演变
