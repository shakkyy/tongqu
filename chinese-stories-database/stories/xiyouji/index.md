# 🐒 西游记故事集

## 系列说明

《西游记》是中国四大名著之一，讲述了唐僧师徒四人西天取经的故事。孙悟空、猪八戒、沙和尚保护唐僧，一路上降妖除魔，历经九九八十一难，终于取得真经。这里的故事适合儿童阅读，语言生动有趣，情节简单明了。

## 阅读建议

1. **按顺序阅读**：从石猴出世开始，了解孙悟空的身世
2. **角色扮演**：和孩子一起扮演师徒四人，表演故事
3. **讨论道理**：每个故事都蕴含道理，引导孩子思考
4. **看动画片**：配合《西游记》动画片一起看，加深印象
